# BaiduYunEnhancer

## 1、功能
通过修改浏览器的操作系统和（或）硬件平台（navigator.platform），破解百度云/百度网盘的下载限制。
* ① 支持直接使用浏览器或迅雷等下载百度云文件；
* ② 支持直接下载整个文件夹；
* ③ 支持选择多个文件一次性批量下载。
